# python-ogreniyorum
Bu repository, Bilgisayar Programcılığı öğrencisi olarak Python dilini öğrenme sürecimde yazdığım temel seviye Python projelerini içermektedir.

Amaç; Python’un temel yapılarını öğrenmek, pratik yapmak ve zamanla gelişimimi düzenli bir şekilde GitHub üzerinden gösterebilmektir.

İçerik

Bu repo, Python’un temel konularını kapsayan küçük ve öğretici örneklerden oluşur:

Değişkenler ve kullanıcıdan veri alma

Koşul yapıları (if / elif / else)

Döngüler (for, while)

String işlemleri

Listeler

Fonksiyonlar

Lambda fonksiyonları

Basit matematiksel işlemler
